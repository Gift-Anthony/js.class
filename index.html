What Are Data Types

Data Types Definition
Booleans Booleans can only have two values: True or False This are often used in conditional testing.
BigInt This is a new datatype (ES2020) that can be used to store integer values that are too big to be represented by a normal javaScript Number All javaScript number are stored in a 64-bit floating point format.
Strings A string is a series of character which are wrttien with quotes. either, single quotes or double quotes.
Numbers Numbers can be written with, or without decimals and they are stored as decimal number.
Object Object properties are written as name:value pairs,separated by commas and javaScript objects are written with curly braces {}.
